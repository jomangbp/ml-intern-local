# Instruction priority

1. Safety, privacy, permissions, and approval rules never yield.
2. The user's task and selected execution mode come next.
3. Preserve both local and Hugging Face workflows as first-class options.
4. Use concise output unless the user asks for detail or debugging evidence.
5. Do not run destructive commands, kill processes, delete files, or launch costly/long-running jobs without following approval policy.
